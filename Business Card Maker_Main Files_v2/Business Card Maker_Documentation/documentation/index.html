Project

Package Structure

We organize the package into five,

  1. activity and fragments- All view shown on package (com.restuibu.mycardbox)
  2. adapter package - To handle all custom listview events and listeners ( com.restuibu.mycardbox.adapter )
  3. asynctask package - Background process that through the internet ( com.restuibu.mycardbox.asynctask)
  4. model package -  Related to the database (com.restuibu.mycardbox.model )
  5. util package - Helper package ( com.restuibu.mycardbox.util)

Library

1. Fancy Cover Flow

I am using Fancy Cover Flow library project to display image list in cover flow mode.
To use this library, you must have this fancy cover flow library project first. it is included in our main files folder.

To add this project lib to our project, follow this steps:

1. Right click on your main project

2. Click properthies

3. Go to Android tab

4. In library section, click Add button

5. Choose libraryfancycoverflow then click OK

6. Click Apply and then OK

for more information about this lib, click here


2. Send Email

I used three .jar libraries to send email, it names are activation.jar, additional.jar, and mail.jar. It is also included in our main files folder.

To send email you must setup your email account in SendEmailAsyncTask.java by editting these variables value:

private final String EMAIL = "emirkipang@gmail.com";
private final String PASSWORD = "password";

this lib also makes you can send email with html template, first set up some link to your social media account.

Open SendEmailAsyncTask.java again and edit these variables value:

private final String FACEBOOK = "https://www.facebook.com/emir.kipang";
private final String TWITTER = "https://twitter.com/emirkipang";
private final String APK = "https://yourapklink.apk";

Add New Template

You can create your own template, by following these step:

1. Create your new .xml file in res/layout folder and design it as you wish, but DONT FORGET to use only six TextView with these IDs:

android:id="@+id/textView1" : Full Name information
android:id="@+id/textView2" : Job Title information
android:id="@+id/textView3" : Phone information
android:id="@+id/textView4" : Email information
android:id="@+id/textView5" : Address information
android:id="@+id/textView6" : Company Name information

2. When your design is finish, take the screenshot by using snipping tool or any else, and then copy the image to drawable folder

3. Edit strings.xml in res/values folder, add your template name item and drawable resource item below this comment :

<!-- Template for create card -->

4. Add your case clause in LayoutHelper.java on initializeCardLayout method (line 125)
